# Miss LTS

The idea of this repository is managing source code for both frontend and backend implementation. The structure will be:

```
.
|-- .github/
|   |-- workflows/
|   |   |-- actions.yml
|-- .gitignore
|-- frontend/
|   |-- build/
|   |-- src/
|   |-- package.json
|   |-- package-lock.json
|-- backend/
|   |-- build/
|   |-- src/
|   |-- package.json
|   |-- package-lock.json
|-- configs/
|   |-- server.cfg
|   |-- rules.sh
|-- scripts/
|   |-- build.sh
|   |-- deploy.sh
|   |-- validate.sh
|-- README.md
```
